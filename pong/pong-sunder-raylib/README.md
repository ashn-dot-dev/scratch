# Pong

## Dependencies
+ [raylib](https://github.com/raysan5/raylib) 4.5.0.
+ [raylib-sunder](https://github.com/ashn-dot-dev/raylib-sunder) bindings for raylib 4.5.0.

## Building
### Linux
```sh
$ make build
```

## Running
### Linux
```sh
$ ./pong
```

### Linux (Pinebook Pro)
```sh
$ LIBGL_ALWAYS_SOFTWARE=true ./pong
```
